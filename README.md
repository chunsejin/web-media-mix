# 미디어 믹스란?
사전적 의미는 신문이나 라디오, 잡지 또는 텔레비전과 같은 서로 다른 성격의 여러 가지 광고 매체를 적절하게 혼용하여 광고 효과를 극대화하려는 전략이다.

# 메타 데이터란?
문자적 의미로는 데이터에 대한 데이터(data about data)라고 정의되나 일반적으로 다양한 업무를 지원하기 위하여 사용되는 정보자원에 대한 구조화된 데이터를 의미하는 것으로 해석된다.

# 현재 단계 및 향후 방향
